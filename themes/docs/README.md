# atoll-theme-docs

Official **docs** theme for `atoll-cms`.

## Install

```bash
php bin/atoll theme:install https://github.com/atoll-cms/atoll-theme-docs/archive/refs/heads/main.zip
php bin/atoll theme:activate docs
```

## Scope

- `assets/main.css` provides the documentation-focused visual style.
- `templates/` provides docs layout/header/footer overrides.
